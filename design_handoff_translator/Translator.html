Translator · Hi-fi prototype

A calm place to ask "what's going on?"

A single-screen AI app for older adults. You type or speak in your own words; you get back something clear and copyable. No accounts, no settings, no jargon. The pipeline that classifies, optimizes, and flattens the response is hidden by design — every interaction is one shot, one button, one answer.

01 · Tokens

Warm off-white as the page, deep ink as the type. One trusted accent (forest green) for action and links. Cream for the answer card so it feels like something handed to you. Terra-cotta is reserved — it only appears on a VERIFY response when the system is telling you something is a scam.

Live mobile prototype

02 · iPhone · interactive

Tap any of the eight scenario chips at the bottom (DECODE, VERIFY, RESPOND, COMPOSE, EXPLAIN, PLAN, DECIDE, CREATE) to run a realistic flow end-to-end. Or type freely, tap "Show me what to ask" for starters, or use the simulated voice button. The "Help me" button kicks the (mocked) pipeline; after ~2.4s the answer card lands.

Why this color, not blue? Tech blue would push the app toward "AI assistant" expectations. Forest green reads closer to civic and financial trust — a notarized document, a savings bond — without drifting into the AARP-beige cliché.

Key answer states

04 · Tonal variation

Three answer-card tones — never shown to the user as labels, only felt. Calm cream is the default for explanation, planning, decisions. Warm parchment is for correspondence. Alert terra-cotta is only ever used for VERIFY responses when fraud is detected.

DECODE · Calm cream Medicare letter, premium increase
VERIFY · Alert terra-cotta IRS gift-card scam — "this is a scam"
COMPOSE · Warm parchment Thank-you note to a neighbor
Notes for next pass. Real backend wiring (Anthropic SDK, Upstash logging, Web Speech). A non-mocked first-run state where examples are hidden until the user pauses for >1.2s. An "I changed my mind" path on the answer screen for nudging a reply tone (shorter / warmer / more formal). And a print-friendly answer view, for the users who will actually print this and put it in a folder.
